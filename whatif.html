# Player ELO ELO +/- Rank +/- Cups Avg Best
How it works: For each cup, all players eliminated in the first N rounds are removed. Remaining players get renumbered positions (with ties for same-round eliminations). Weighted ELO is recalculated from scratch across all cups. This shows who performs best when early-round variance is removed.

Why? Early rounds can be chaotic — one bad start and you're out. Skip rounds reveals who consistently performs in mid-to-late game elimination.
